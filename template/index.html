Design Purposes

This operating system is spining mode of Yuros Blackbird Project. Specially designed by Member of Yuros Chapter UIN Syarif Hidayatullah Jakarta. The goal of this project is academic activities in Technology Courses at UIN Syarif Hidayatullah Jakarta, Department of Library and Information Science.

The system provides a lightweight and customizable environment focused on digital preservation, archive management, metadata processing, and digital library services. It includes various open-source tools and command-line utilities commonly used in library and archival workflows.

One of the main features of this operating system is Terminal Recording during the installation process using CLI (Command Line Interface). This feature automatically records installation activities and terminal commands for learning, documentation, troubleshooting, and academic evaluation purposes.

Main Features

  • CLI-based installation process
  • Automatic terminal session recording
  • Lightweight and optimized Linux environment
  • Educational-focused system configuration
  • Open-source and customizable platform

Educational Purpose

This operating system is developed specifically for teaching and learning activities in Digital Library and Archive courses. It is intended to help students understand:

  • Linux-based system administration
  • Develop Digital preservation workflows
  • Develop Digital library model
  • Develop Archive management systems
  • Documentation and system logging
